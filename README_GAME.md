  - Getting out with this fragment gives it to you
  - Getting enough of these fragments will unlock a new area on the world map with special stages?

**Stage Layout**

- [ ] Large, room-split
  - Large stages split into multiple rooms
  - Player would have either a set number of lives for the entire stage or just unlimited
  - On entering a room, the respawn point is set at that entrance
  - Benefits:
    - Cooler stage design, less need to pack spikes absolutely everywhere
    - Collectibles can be sparsely spaced out (easier to make hidden secrets)
    - Future gamemodes can be added like roguelike
    - Could be better for performance since the max room size can be smaller
    - Story telling is easier
  - Drawbacks:
    - Far more complex, both code and editor
    - Lots more art since stages must be filled with something

**Interesting Features**

- [ ] Cursed tablet
  - Picking up causes the player to die after x seconds or something unless they reach a purifying shrine in time
  - _Large room-split_ works very well with this since you would need to find the shrine room and the tablet room and then do it, which is neat.
- [ ] Temple Idol Traps
  - Picking up the Goal, causes the temple to begin to break down and become harder, the player must then get to an exit without dying
  - Benefits:
    - Far more interesting, the stage has 2 phases now
  - Drawbacks:
    - Complexity, not even sure how to implement or what would change
